# 💰 Expense Tracker

A simple and interactive Expense Tracker web application built using **HTML, CSS, and JavaScript**. It helps users manage their income and expenses, track their balance, and save transaction history using Local Storage.

## 🔗 Live Demo

**View Live Demo**

> https://reethikasrilatha-lab.github.io/Expense-Tracker/

---

## 📌 Features

- Add income and expense transactions
- Automatically calculates total balance
- Displays total income and total expenses
- Delete unwanted transactions
- Stores data using Local Storage
- Clean, responsive, and user-friendly interface
- Built with vanilla JavaScript (no external libraries)

---

## 🛠️ Tech Stack

- **HTML** – Structure of the Expense Tracker
- **CSS** – Styling, layout, and responsive design
- **JavaScript** – Transaction management, calculations, DOM manipulation, and Local Storage

---

## 🚀 How It Works

1. Enter a transaction description.
2. Enter the transaction amount.
3. Select **Income** or **Expense**.
4. Click the **Add Transaction** button.
5. View the updated balance, income, and expense totals instantly.
6. Delete any transaction when needed.
